# 血统系统（Bloodline）

## 目标
通过可计算指标描述角色状态，并驱动能力可用性与剧情风险。

## 核心指标
1. `purity`（纯度，0-100）
- 影响等级评估与高阶能力上限。

2. `stability`（稳定性，0-100）
- 影响高压场景中的可控性。
- 低稳定状态下，高阶能力可能被禁用。

3. `corruption`（腐蚀，0-100）
- 反映失控风险与命运代价。
- 高腐蚀将触发限制、异化或剧情惩罚。

## 风险阈值（MVP建议）
- 稳定性 < 30：进入高危区，限制高阶言灵。
- 腐蚀 >= 60：进入预警区，触发强制风险提示。
- 腐蚀 >= 80：进入禁用区，禁止推进某些破坏性支线。

## 剧情触发关系
- 高纯度 + 低稳定：常见“力量可用但代价陡升”。
- 中纯度 + 高稳定：常见“稳态推进”路线。
- 高腐蚀：触发宿命压力与关系破裂事件。

## 设计约束
- 任何血统变化必须有触发原因与叙事后果。
- 不允许无来源的数值跃迁。
