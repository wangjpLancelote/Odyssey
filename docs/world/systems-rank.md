# 等级系统（Rank）

## 目标
定义角色在战术行动与叙事承担中的能力层级，作为“可做什么”的第一道门槛。

## 等级划分
- D：观察/辅助层；不具备高压正面作战资格。
- C：基础执行层；可参与低风险任务。
- B：标准执行层；可独立处理常规高压事件。
- A：核心执行层；具备高阶协同与关键任务资格。
- S：极限个体层；需严格监管与特殊约束。

## 晋升判据（MVP）
1. 血统纯度达到阈值区间。
2. 稳定性在安全范围内。
3. 实战记录与关键选择一致（不是只看战力）。
4. 无严重规则违规历史。

## 行为上限（示例）
- B 级可稳定使用中阶言灵，默认不可无代价使用高阶言灵。
- A/S 可触达更高输出，但腐蚀风险与剧情代价同步上升。

## 叙事后果
等级越高，默认承担更高责任与更重心理代价，避免“高等级=无成本特权”。
