# 术语表

## 核心术语
- 世界观基石：项目的顶层叙事与规则规范，优先级高于单章创作。
- 主线：固定章节骨架，不可被 AI 改写。
- 支线：在规则边界内生成的可选叙事路径。
- 足迹：玩家私有的已达成路径记录与检查点集合。
- 回溯：从检查点恢复状态；不等于删除历史后果。

## 系统术语
- 等级（Rank）：角色能力层级（D/C/B/A/S）。
- 血统（Bloodline）：`purity/stability/corruption` 三维状态。
- 言灵（Word Spirit）：具门槛、冷却、代价的能力系统。
- 章节时间线：`storylineId + ordered chapters` 的推进约束。
- 安全回退：命中世界观冲突后的保底转向路径。

## 文档术语
- 硬规则：不可违背设定。
- 软规则：可调整表达，但需记录理由。
- 规则 ID：用于检测与回退映射的稳定标识符。
