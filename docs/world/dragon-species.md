# 龙种体系

## 起源
龙族（Dragon）是地球上最古老的智慧物种，远早于人类文明。龙族拥有超越人类的力量、寿命与智慧，曾统治整个星球。随着人类崛起与龙族内部分裂，龙族数量急剧衰减，残存者或沉睡、或隐匿于人类社会。

## 龙种分类

### 1. 纯血龙族（Pure-blood Dragon）
- 完整的龙族基因，能够在人形与龙形之间切换。
- 拥有极强的再生能力、超长寿命与高阶言灵。
- 现存数量极少，多数处于沉睡或隐匿状态。
- 纯血龙族之间存在严格的血脉等级：龙王 > 君主 > 普通纯血。

### 2. 混血种（Hybrid / 龙裔混血）
- 龙族与人类的后代，携带不同纯度的龙族血统。
- 血统纯度（purity）决定能力上限与觉醒可能性。
- 大多数混血种在日常状态下与普通人无异，只在特定条件下觉醒。
- 卡塞尔学院的学生绝大多数属于此类。
- 等级划分：S / A / B / C / D（基于血统纯度与觉醒潜力）。

### 3. 死侍（Dead Servant）
- 被龙族精神控制或寄生的人类。
- 失去自主意志，成为龙族的傀儡与耳目。
- 外表与常人无异，极难辨识。
- 龙王级别的死侍可能保留部分智能，执行复杂任务。
- 死侍是龙族渗透人类社会的主要手段。

### 4. 寄灵人
- 被龙族灵魂片段寄宿的人类。
- 与死侍不同，寄灵人可能保留部分自我意识。
- 龙族灵魂与宿主之间存在博弈关系。
- 是剧情中"身份认同"冲突的核心载体之一。

## 龙族形态

### 人形态（拟态）
- 纯血龙族可以完美拟态为人类外貌。
- 维持拟态消耗极少，可长期保持。

### 龙形态（完全体）
- 展现龙族本来面目，战斗力远超人形态。
- 完全体形态下言灵威力大幅提升。
- 现代极少有龙族展现完全体（暴露风险极高）。

### 半龙化
- 混血种在血怒或高压状态下可能出现局部龙化特征。
- 表现为：鳞片浮现、瞳孔竖裂（黄金瞳）、局部硬化等。
- 半龙化是失控前兆，稳定性急剧下降。

## 与游戏系统的映射
| 设定概念 | 对应系统 | 说明 |
|---------|---------|------|
| 血统纯度 | `bloodline.purity` | 决定等级上限与能力天花板 |
| 龙化稳定性 | `bloodline.stability` | 低稳定触发失控风险 |
| 龙族侵蚀 | `bloodline.corruption` | 龙族血统对人类意识的吞噬程度 |
| 等级评定 | `rank` (D-S) | 混血种的综合战力分级 |
| 龙族形态切换 | 剧情事件 | 通过主线/支线节点触发，非玩家主动操作 |
