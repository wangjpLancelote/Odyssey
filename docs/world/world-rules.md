# 世界规则

## 硬规则（不可违背）
1. 主线时序固定
- 章节推进必须遵循时间线配置。
- 禁止跳章、逆章、环形推进。

2. 能力边界固定
- 等级、血统、言灵三系统共同决定可行动作上限。
- 不允许“无前置条件”释放高阶能力。

3. 认知边界固定
- 角色知识受时间线门控。
- 未触达事件不允许提前知晓关键真相。

4. 后果记忆固定
- 足迹与检查点用于恢复状态，不用于“擦除历史”。
- 回溯后仍需保留分支痕迹与已发生事实。

5. 世界观优先级最高
- 世界规则 > AI 支线创作 > 风格化表现。
- 出现冲突时必须回退到安全分支。

## 软规则（可变但需受控）
1. 角色语气可变：允许同一角色在高压场景出现强烈语气漂移。
2. 场景风格可变：可采用夸张分镜表达，但不得改变事件事实。
3. 支线密度可调：可根据调参系统调整触发率，不得突破硬规则。

## 规则冲突处理
1. 检测：生成结果先经过规则检查。
2. 拦截：命中硬规则冲突即阻断落库。
3. 回退：转入预定义安全节点，记录冲突 ID。
