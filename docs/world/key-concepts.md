# 关键概念

## 黄金瞳（Golden Eyes）
- **定义**：龙裔血统觉醒的外在标志，瞳孔变为金色竖瞳。
- **触发条件**：
  - 极端情绪刺激（恐惧、愤怒、保护欲）。
  - 血统纯度突破临界阈值。
  - 外部龙族力量的共鸣或压迫。
- **表现**：瞳孔从人类形态变为金色竖裂瞳，伴随感官增幅、反应速度提升。
- **风险**：黄金瞳出现意味着 `stability` 急剧下降，是血怒状态的前兆。
- **剧情意义**：ch02 标题即为"黄金瞳"，标志着主角首次经历血统觉醒。
- **游戏映射**：黄金瞳事件为剧情节点触发，伴随 `bloodline.stability` 骤降与 `corruption` 风险提示。

## 血怒（Dragon Rage）
- **定义**：龙裔血统失控后的暴走状态。
- **机制**：
  - 龙族血液中的本能压过人类理智。
  - 战斗能力暴增，但完全丧失理性判断。
  - 持续时间越长，回复越困难，永久伤害风险越高。
- **等级**：
  - **初级血怒**：短暂失控，可被外力唤醒（如同伴呼唤）。
  - **深度血怒**：长时间暴走，可能造成不可逆的身体/精神损伤。
  - **完全龙化**：极端情况下触发完全体变异，通常意味着人格死亡。
- **游戏映射**：
  - `stability < 30` 时进入血怒高危区。
  - 血怒状态下言灵威力 ×2 但 `corruption` 快速上升。
  - 触发血怒后必须经历"代价节点"（符合三支柱中的"自我选择"）。

## 尼伯龙根（Nibelungen）
- **定义**：龙族创造的异空间/结界，独立于现实世界的口袋维度。
- **特征**：
  - 每个尼伯龙根由创造它的龙族个体定义规则。
  - 内部物理法则可能与现实不同。
  - 时间流速可能与外界不一致。
  - 是龙族沉睡、藏宝、设伏的主要场所。
- **危险性**：
  - 进入尼伯龙根后与外界隔绝，通讯中断。
  - 龙族在自己的尼伯龙根内战力大幅提升。
  - 出口可能被封锁，形成"密室死斗"。
- **剧情功能**：高压战斗场景的核心舞台，强制"孤独"主题——进入后无外援可依。
- **游戏映射**：尼伯龙根内的场景属于封闭剧情区间，回溯点在进入前自动创建。

## 龙文（Dragon Script）
- **定义**：龙族的古老文字系统，承载言灵与龙族知识。
- **特征**：
  - 非人类语言，无法通过常规学习掌握。
  - 龙裔可能本能地识别部分龙文（取决于血统纯度）。
  - 高阶言灵的释放本质上是"说出"对应的龙文。
- **剧情功能**：解密与知识揭示的载体。破译龙文通常意味着解锁新信息或触发隐藏事件。

## 屠龙术（Dragon Slaying Arts）
- **定义**：人类/混血种对抗龙族的系统化战术与技术体系。
- **体系**：
  - 由卡塞尔学院系统化整理与传授。
  - 结合了现代军事技术与龙裔特殊能力。
  - 针对不同类型龙族有专门的战术方案。
- **核心原则**：以弱胜强，利用龙族的已知弱点与团队协作弥补个体差距。

## 秘党执照（License）
- **定义**：秘党颁发的龙裔行动许可证。
- **等级**：与龙裔等级对应，限定可执行的任务级别与可使用的资源。
- **意义**：龙裔在人类社会中行动的"合法性"来源。无执照行动视为违规。

## 君主制约（Monarch's Binding）
- **定义**：龙王或君主级别龙族对低阶龙裔施加的精神压制。
- **表现**：
  - 低阶龙裔在君主面前可能无法行动。
  - 血统纯度越高，被压制的概率越大（因为龙族血液更"听从"上位者）。
  - 是路明非 S 级血统的双刃剑——力量更强，但也更容易被龙王控制。
- **游戏映射**：在与君主级敌人接触的节点中，`purity` 越高可能触发特殊限制事件。
