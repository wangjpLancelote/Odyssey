# 三支柱主题

## 1) 孤独
孤独不是背景情绪，而是持续施压机制：
- 角色在关键节点必须承受“无法完全被理解”的状态。
- 关键选择前后，系统通过对白、镜头、音频体现心理距离感。
- AI 支线不得将核心冲突简化为“外部角色立即救场”。

## 2) 自我选择
角色价值通过“主动承担后果”成立：
- 每个核心节点至少存在一项“主动承担代价”的选项。
- 选择应体现角色价值偏好（忠诚、求生、责任、逃离）。
- 回放可复盘选择，但不抹除其后果记录。

## 3) 宿命选择
宿命不是取消选择，而是限制选择空间：
- 主线必经节点固定，保证时代与世界线不被改写。
- 支线可扩展情绪与信息，不可改变主线骨架。
- 任何“逆天改命”表达必须付出明确成本并受规则拦截。

## 落地要求（MVP）
- 文本：每章至少一次“三支柱同场”节点（情绪 + 抉择 + 代价）。
- 分镜：关键抉择前后必须有镜头语法差异（推拉/留白/冲击）。
- 音频：抉择前后至少一个可辨识的氛围层变化。
