# 组织与角色关系

## 组织结构（风格改编）
1. 学院系统
- 负责训练、筛选与知识保全。
- 强调纪律与任务优先级。

2. 执行体系
- 负责外勤处置与高压事件响应。
- 与学院在“风险承受”上常有分歧。

3. 灰幕势力
- 活动在正式体系外。
- 提供信息、资源或诱导，推动道德困境。

4. 民间血裔圈层
- 与主体系保持松散联系。
- 常作为支线冲突与价值对照来源。

## 角色关系张力
1. 责任 vs 自保
- 决策常在“保护他人”与“保全自己”之间撕扯。

2. 服从 vs 自主
- 命令体系与个人判断存在持续冲突。

3. 信任 vs 利用
- 合作关系可能随真相揭示而重构。

## 冲突边界
- AI 生成支线可强化关系张力，但不得颠覆主线核心角色定位。
- 不允许把核心人物降格为无动机工具人。
