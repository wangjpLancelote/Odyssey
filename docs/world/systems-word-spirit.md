# 言灵系统（Word Spirit）

## 目标
把高冲击能力纳入可验证规则，避免“设定外开挂”。

## 分层
- Tier 1-3：低阶，适合常规控制与辅助。
- Tier 4-6：中阶，具备明确战术优势。
- Tier 7-10：高阶，具备战略级影响，需严格门槛。

## 使用条件
1. 角色等级满足最低门槛。
2. 血统稳定性满足安全阈值。
3. 冷却与消耗满足当前状态。
4. 场景允许该能力类型（不违背世界规则）。

## 代价模型
- 每次释放必须体现至少一类代价：
  - 生理负担
  - 心理负担
  - 战术暴露
  - 关系后果

## 禁用边界
1. 明确禁止“连续无代价高阶释放”。
2. 禁止在认知未解锁时释放剧情关键型能力。
3. 禁止绕过主线必经节点直接改写章节结局。

## MVP内置示例
- 君焰（高风险高回报）
- 镰鼬（机动与感知倾向）
