# 龙王与君主

## 龙王（Dragon Kings）

龙王是龙族中最强大的存在，代表龙族意志的两极。龙族的命运围绕两位龙王的对立展开。

### 黑王·尼德霍格（Nidhogg, the Black King）
- **立场**：毁灭。认为人类是威胁，必须被消灭或奴役。
- **状态**：活跃（或周期性苏醒）。
- **能力**：
  - 精神控制：可大规模制造死侍。
  - 言灵压制：能压制低阶龙裔的言灵释放。
  - 时间感知：对历史与因果链拥有超常感知。
- **剧情角色**：终极对立面。黑王的意志贯穿所有故事线，即使未直接登场，其影响也通过死侍、龙族遗迹、阴谋链条传导到每一章。
- **游戏映射**：黑王势力是 `corruption` 增长的主要来源；与黑王相关的选择往往伴随高风险高回报。

### 白王（The White King）
- **立场**：共存。选择与人类和平相处，相信混血种是两族未来的桥梁。
- **状态**：沉睡。白王在远古时代选择沉睡，将意志寄托于后代与信念。
- **遗产**：
  - 卡塞尔学院的创建理念源于白王的共存思想。
  - 部分高阶言灵被认为是白王血脉的馈赠。
- **剧情角色**：精神灯塔。白王不直接干预事件，但其选择——"即使沉睡也不毁灭"——是主角面对宿命时的价值锚点。
- **游戏映射**：白王路线倾向维持 `stability`、控制 `corruption`，代价是放弃部分力量峰值。

## 四大君主（Four Monarchs）

四大君主是仅次于龙王的纯血龙族，各据一方，拥有独立势力与意志。

### 1. 格劳科斯（Glaucus）
- **别称**：海之君主。
- **领域**：海洋与深渊。
- **特征**：极端的隐匿能力，长期潜伏于深海。
- **与主线关系**：火之晨曦 MVP 中未直接登场，但其势力范围可能在后续章节涉及。

### 2. 尼德霍格的将领们
- 黑王麾下的高阶纯血龙族，执行黑王意志。
- 在不同时代以不同身份渗透人类社会。
- **与主线关系**：是卡塞尔学院主要对抗的目标。

### 3. 诺顿（Norton）
- **身份**：与黑王有深层关联的关键角色。
- **特征**：以学生身份潜伏于卡塞尔学院。
- **剧情功能**：身份揭示是重大转折点，涉及信任崩塌与阵营重构。
- **与主线关系**：在火之晨曦中以隐藏身份存在，是后续章节的关键引爆点。

### 4. 其他沉睡者
- 远古时代选择沉睡的纯血龙族。
- 分布在全球各地的龙族遗迹中。
- 一旦被唤醒，将带来区域性灾难事件。
- **游戏映射**：支线事件的潜在触发源。

## 龙王对立与剧情驱动

```
黑王（毁灭）◄──────────►白王（共存）
     │                        │
     ▼                        ▼
  死侍/阴谋链              卡塞尔学院
  corruption ↑             stability ↑
  力量峰值高               可控性高
     │                        │
     └──────── 玩家选择 ────────┘
```

玩家在主线推进中不断面对两极之间的抉择：
- 接受更多龙族力量（purity↑ corruption↑）→ 短期强大但失控风险升高
- 维持人类意志（stability↑ corruption↓）→ 可控但能力受限

这一对立是三支柱主题"宿命选择"的核心载体。

## 火之晨曦 MVP 中的呈现
- ch01（卡塞尔之门）：黑王与白王仅以背景知识出现，通过学院入学引导间接提及。
- ch02（黄金瞳）：黑王势力的痕迹开始显现，玩家首次感受到龙族阴影的压迫感。
- 后续章节：龙王对立将从背景升格为直接冲突驱动力。
