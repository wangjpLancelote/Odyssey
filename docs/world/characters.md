# 核心角色档案

## 主角

### 路明非（Lu Mingfei）
- **身份**：卡塞尔学院新生，表面上是普通的中国高中毕业生。
- **血统等级**：S 级（极罕见，初期未被完全评估）。
- **言灵**：待觉醒（S 级血统潜力巨大，但初期未激活稳定言灵）。
- **性格**：自嘲、怯懦表象下藏着极强的求生本能与共情能力。习惯用玩世不恭掩饰内心恐惧。
- **核心冲突**：
  - 不相信自己是"被选中的人"，但不断被推向命运中心。
  - S 级血统意味着巨大力量，也意味着最高的失控风险。
  - 在"做普通人"与"承担龙裔宿命"之间持续撕扯。
- **剧情功能**：玩家的代入锚点。路明非的选择就是玩家的选择。
- **游戏映射**：初始 rank=D（潜力未激发），purity 极高但 stability 波动大。

### 楚子航（Chu Zihang）
- **身份**：卡塞尔学院精英学生，实力顶尖。
- **血统等级**：A 级。
- **言灵**：君焰（Sovereign Flame）— 极高破坏力的火系言灵；雷切（Raikiri）— 雷电高速斩击。战斗风格偏近战精准打击。
- **性格**：沉默寡言、自律极致、近乎苦行僧式的生活方式。内心背负沉重的过去。
- **核心冲突**：
  - 为了保护重要的人愿意付出一切，但情感表达极度压抑。
  - 对自身龙族血统的排斥与接受之间的博弈。
- **剧情功能**：与路明非形成鲜明对照——一个回避命运，一个主动承受命运。
- **与主角关系**：从冷漠到信任的搭档关系，是主线中最重要的同伴线。

### 凯撒·加图索（Caesar Gattuso）
- **身份**：卡塞尔学院学生会主席，出身显赫的龙裔家族。
- **血统等级**：A 级。
- **言灵**：镰鼬（Kamaitachi）— 风刃攻击与感知增幅。
- **性格**：自信、领袖气质、骄傲但并非自私。有强烈的责任感与家族使命感。
- **核心冲突**：
  - 家族期望与个人判断的冲突。
  - "为多数人牺牲少数人"的功利主义 vs 路明非的"不放弃任何人"。
- **剧情功能**：权力与责任的具象化。他的选择代表"体制内的正确答案"，与主角的直觉式选择形成张力。

### 诺诺 / 陈墨瞳（Nono / Chen Motong）
- **身份**：卡塞尔学院学生，身份背景复杂。
- **血统等级**：A 级。
- **性格**：外表温柔开朗，内在坚韧。对路明非有特殊情感。
- **核心冲突**：
  - 真实身份与公开身份之间的裂痕。
  - 在忠诚与自保之间的选择。
- **剧情功能**：信任主题的核心载体。她的身份揭示是"信任 vs 利用"张力的引爆点。

## 关键配角

### 昂热 / 安吉（Ange）
- **身份**：卡塞尔学院校长。
- **血统**：极其神秘，可能是目前已知最强的混血种之一。
- **言灵**：时间零（Time Zero）— 能局部操控时间的禁忌级言灵。
- **性格**：温文尔雅的外表下隐藏着深不可测的算计与悲悯。永远在棋局中思考。
- **剧情功能**：导师与操盘手。昂热的指引推动主线进展，但他的真实意图始终是谜。
- **与主角关系**：招募路明非进入学院的人，既是保护者也是利用者。

### 芬格尔（Fingal）
- **身份**：路明非的室友，学院中的"活宝"。
- **血统等级**：C 级。
- **性格**：话痨、乐观、看似不靠谱但关键时刻有担当。
- **剧情功能**：日常叙事的润滑剂，降低高压剧情的窒息感。

### 源稚生（Minamoto Chishou）
- **身份**：蛇岐八家出身，日本龙裔世家的继承人。
- **血统等级**：A 级。
- **言灵**：镰鼬（Kamaitachi）— 风刃/感知系。
- **性格**：谦和有礼，内藏锋芒，行事风格兼具东方武道精神与政治敏感。
- **剧情功能**：蛇岐八家与卡塞尔学院之间关系的纽带，也是"东西方龙裔冲突"的缩影。

### 绘梨衣（Eri）
- **身份**：日本少女，与火之晨曦主线深度绑定。
- **特征**：与龙族有特殊联系，身份与命运极为悲剧。
- **剧情功能**：火之晨曦故事线的情感核心。她的命运直接影响主角的关键抉择。
- **与主角关系**：路明非在火之晨曦篇中最重要的情感羁绊。

### 诺顿（Norton）
- **身份**：学院学生（公开），真实身份与黑王密切关联。
- **血统**：极特殊，非普通混血种。
- **性格**：表面友善随和，实际冷酷理性，有独立于黑王的个人意志。
- **剧情功能**：卧底与身份反转的载体。他的立场揭示将重塑玩家对前期所有互动的理解。
